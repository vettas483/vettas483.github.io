# Сайт о фильмах
![alt text](assets/images/preview.png)

## Архитектура сайта

**Сайт включает в себя следующие страницы:**

1. **Главная** - показывает описание и связь с автором
2. **Фильмы** - отоброжает список фильмов с описанием, при нажатии переходит на сайт к просмотру


## Запросы для получения данных

**Ссылка на API - `http://127.0.0.1:3000/final.%20Personal/film.html`**

## Используемые технологии

1. `HTML5`
2. `CSS3`
3. `JavaScript Vanilla`
